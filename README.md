- A PostgreSQL Instance

### Installing 

Clone the repository

`git clone https://github.com/itsrennyman/aurora.git`

Install the dependencies

`npm install`

Copy & Fill the *.env.example*

`cp .env.example .env`

Start the server

`npm run dev`
